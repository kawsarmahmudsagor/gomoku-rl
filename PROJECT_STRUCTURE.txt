🎯 TRAINING SCENARIOS
════════════════════════════════════════════════════════════════════════════

Scenario 1: DEFAULT (Recommended)
  python train.py
  • 200k episodes, 30-60 min
  • Balanced training (20% self-play)
  • ~60% win rate expected

Scenario 2: EXTENDED (Stronger AI)
  python train.py --episodes 500000 --self-play-ratio 0.4
  • 500k episodes, 2-3 hours
  • Higher self-play ratio
  • ~80% win rate expected

Scenario 3: QUICK TEST
  python train.py --episodes 20000 --checkpoint-interval 2000
  • 20k episodes, 5 minutes
  • For testing/debugging
  • Weak agent (~10% win rate)

Scenario 4: EVALUATION ONLY
  python train.py --eval-only --model-path models/gomoku_agent_final.pt
  • Test existing model
  • No training, just evaluation
  • ~2 minutes for 100 games

🔧 CONFIGURATION OPTIONS
════════════════════════════════════════════════════════════════════════════

Training Hyperparameters:
  --episodes 200000          Total episodes to train
  --learning-rate 1e-4       Adam optimizer step size
  --gamma 0.99               Discount factor
  --buffer-size 100000       Replay buffer capacity
  --checkpoint-interval 10000 Save model every N episodes
  --eval-interval 1000       Evaluate every N episodes
  --self-play-ratio 0.2      Fraction of self-play games
  --no-export                Skip ONNX export
  --eval-only                Evaluate existing model
  --model-path MODEL         Path to existing model
